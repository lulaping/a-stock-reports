② Serenity 框架分析

2.1 超级周期定位

终端判断：人形机器人产业处于 导入期→加速期的转折点。2025年是"技术验证年"，2026年是"小批量产年"，2027-2028年才是"规模化爆发年"。谐波减速器是机器人关节的"心脏"部件，单台人形机器人需要14-28个谐波减速器（视设计而定）。

竞争格局：全球谐波减速器市场哈默纳科一家独大（份额~75%），绿的谐波国内份额第一（~25%），丰光精密的差异化在微型（03/05/08）——这是人形机器人手指/手腕级关节的刚需，绿的谐波的优势在中大型（14/17/20/25/32）。微型谐波是蓝海，竞争格局好；中大型是红海，绿的+昊志+来福激烈竞争。

2.2 供应链栈绘制

终端机器人层：宇树科技（H1全地形） / 智元机器人 / 埃斯顿 / 新松 / 安川电机（四大家族）/ 特斯拉Optimus
关节模组层：减速器 + 伺服电机 + 编码器 → 丰光定位：谐波减速器+电机轴+丝杠
精密加工层：丰光精密（03-32全系列谐波量产 + 精密零部件 + 真空泵主轴）
设备/材料层：数控加工中心、热处理设备、特殊钢材（柔轮材料）

Serenity 判读：丰光精密的谐波减速器处于机器人供应链最关键的"关节"环节。但它在谐波市场整体不是卡点——绿的谐波2025年谐波收入~4亿，丰光可能不到1000万。丰光的差异化价值在微型谐波（03/05/08）的稀缺性，而不是整体谐波的市场份额。

⑤ 综合判定

Serenity 最终观点

丰光精密是北交所人形机器人赛道中最有技术壁垒的标的之一。它在微型谐波减速器（03/05/08）上的"国内唯一量产"地位是真实的卡点——不是因为现在赚多少钱，而是因为当人形机器人爆发时，这种微型高精度减速器的供给极度稀缺。

它好在哪：

  • 微型谐波国内唯一：03/05/08型国内只有丰光能量产，这是人形机器人手指/手腕级关节的刚需部件
  • 顶级客户验证完毕：安川36个月考核通过 + Edwards独家真空泵供应商 = 品质已被国际顶尖客户认可
  • 全系列覆盖：03-32型全系列量产，既能做人形也能做工业机器人，不押注单一场景
  • 三角战略清晰：精密零部件（基本盘）+谐波减速器（增长引擎）+半导体国产化（第二曲线），逻辑完整
  • 资产负债表干净：负债率27%，有安全垫

它不好在哪：

  • 还在亏钱：2025全年亏损，2026Q1仍微亏——48亿市值对应负利润
  • 谐波收入极小：没有单独披露（年报中合并计入产品收入），意味着谐波占总营收比例可能<5%
  • 大客户集中度79%：不健康的集中度，任何一个大客户流失都是灾难
  • PB 11.9x：已高于同行业5.07x中位值的2.3倍，市场已经把"谐波故事"的价格打进去了
  • 绿的不是吃素的：绿的谐波也在研发微型减速器，一旦绿的突破，丰光的"国内唯一"护城河就破了
  • 7月21日解禁：限售股解禁是短期肉眼可见的风险

🟡 Serenity 判定：Watch / 不追高。 卡点评分7/15，在微型谐波细分赛道有真实稀缺性，但当前估值已经把"故事兑现到2027年"的预期包含了。48亿买入意味着你在赌2027年谐波收入>8000万+唯实深蓝并表顺利完成+没有新增竞争者，三个条件缺一不可。

安全边际在哪：如果2026Q2季报（8月）显示单季扭亏且谐波收入环比翻倍，20-22元区间是较好的建仓区域（对应2027E PE~25x）。

红线在哪：如果绿的谐波宣布微型谐波量产，或者Q2仍亏损>200万，需要立即重新评估。

一句话：这是北交所最好的机器人卡点标的，但不是最好的价格。等回调，等Q2数据，再决定。
