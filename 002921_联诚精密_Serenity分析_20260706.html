分析框架：Serenity Mode 1（供应链卡点逆向） | 日期：2026-07-06 | 能力圈：精密制造/汽车零部件/机器人

一、结论（两行）

客观框架：Bear 30-35亿（扭亏失败+并购整合风险+毛利率持续<15%）→ Base 45-50亿（2026扭亏+并购标的并表+机器人概念溢价）→ Bull 65-80亿（业绩兑现+新客户放量+市值翻倍）。当前44.6亿已接近Base上沿，向上空间有限，向下有30%下行风险。

Serenity视角：🔴 Not a Card Point — 联诚精密是一家精密铸造/机加工代工商，处在制造业微笑曲线的最底端。毛利率14%、竞争壁垒低、无垄断属性。并购山东鑫亚的消息驱动涨停，但Serenity不追消息面驱动的炒作，尤其当被收购标的具体财务数据仍未披露时。

二、公司概况：它是做什么的？

联诚精密 ≠ 机器人公司。它是一家精密铸造+机加工代工商。


公司主营各类精密机械零部件的研发、生产和销售，涵盖铸铁、铸铝、精密加工全链条。产品形态包括液压行走马达壳体、减速器壳体、液压阀座、曲轴箱体、轴承座、悬挂支架、汽车水泵壳体、工业机器人底座/大臂/转座/连接体等。


产业链定位：

  • 上游：生铁、铝锭、钢材等基础原材料
  • 自身：铸造（铁/铝）→ 精密加工 → 表面处理 → 检测，全链条代工
  • 下游：汽车（含新能源）、工程机械、商用空调、液压机械、工业机器人

产能规模：年产零部件1500万件，年铸造能力30万吨。在山东济宁（总部）、日照、安徽宣城、江苏南京、美国威斯康星、越南胡志明设有生产基地。


关于"机器人概念"：公司确实生产机器人结构件（底座、大臂、转座、连接体），但这些是机器人的结构件而非核心零部件（减速器、电机、丝杠、传感器）。结构件加工壁垒远低于核心零部件，可替代性强。把联诚精密的"机器人概念"等同于绿的谐波/恒立液压的机器人属性，是严重的概念泛化。

四、Serenity 框架评估

4.1 供应链卡点定位

精密铸造+机加工在机器人产业链中的位置：结构件加工（非核心）


机器人产业链价值分布（单台价值量）：

  • 丝杠：~5000-6000元 ← 核心卡点
  • 减速器：~3000-4000元 ← 核心卡点
  • 电机：~2000-3000元 ← 核心卡点
  • 传感器：~1500-2500元 ← 核心卡点
  • 结构件（底座/大臂/壳体）：~1000-2000元 ← 非卡点

核心判断：联诚精密生产的是机器人的"外壳"和"骨架"，不是"关节"和"神经"。结构件加工在中国是一个极度竞争的行业——数千家铸造厂可以生产同样品质的产品。这不是Serenity意义上的供应链卡点。

"精密铸造零部件就像钢筋水泥——哪栋楼都需要，但你不会因为某家钢厂供应了上海中心大厦的钢筋就给它高估值。"
